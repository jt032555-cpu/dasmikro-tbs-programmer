🔧 Complete Wiring Guide

1️⃣ Basic Setup — Receiver → TBS Mini → ESC → Motor + Speaker 

  RC RECEIVER                TBS MINI                   ESC / MOTOR
  ┌──────────┐               ┌──────────────┐           ┌──────────┐
  │  CH1/THR ├─(WHT sig)────▶│ PROP1        │           │          │
  │          ├─(RED +5V)────▶│   +5V        │           │          │
  │          ├─(BLK GND)────▶│   GND        │           │          │
  │  CH2/AUX ├──────────────▶│ PROP2        │           │          │
  │  CH3/AUX ├──────────────▶│ PROP3        │           │          │
  └──────────┘               │              │           │          │
                             │ SERVO OUT1 ──┼──────────▶│ Signal   │
                             │              │           └──────────┘
  BATTERY / BEC              │              │
  ┌──────────┐               │              │           SPEAKER (8Ω)
  │ +5V–9V  ├──────────────▶│ VCC (+)      │           ┌──────────┐
  │ GND     ├──────────────▶│ GND (–)      ├──(SPK+)──▶│          │
  └──────────┘               └──────────────┘    (SPK-)─│          │
                                                         └──────────┘
  ⚠️ IMPORTANT: Connect GND between all devices!

2️⃣ Adding Lights (OUT1–OUT6, low-side switching) 

  BATTERY +  ──────────────[LED]──[330Ω resistor]──┐
                                                    │
  TBS MINI OUT1 ──────────────────────────────────▶─┘ (OUT1 switches to GND)

  Headlights: Battery+ ─▶ LED ─▶ 330Ω ─▶ OUT1 ─▶ GND (internal)
  Tail lights: Battery+ ─▶ LED ─▶ 330Ω ─▶ OUT2 ─▶ GND (internal)
  Left Turn:  Battery+ ─▶ LED ─▶ 330Ω ─▶ OUT3 ─▶ GND (internal)
  Right Turn: Battery+ ─▶ LED ─▶ 330Ω ─▶ OUT4 ─▶ GND (internal)
  Muzzle:     Battery+ ─▶ LED ─▶ 330Ω ─▶ OUT5 ─▶ GND (internal)
  Beacon:     Battery+ ─▶ LED ─▶ 330Ω ─▶ OUT6 ─▶ GND (internal)

  ⚠️ ALWAYS use a resistor (220–470Ω) with LEDs to prevent burning them out!

3️⃣ Programming Setup — USB-to-Serial Adapter → TBS Mini 

  USB-to-Serial Adapter          TBS Mini
  (CP2102 / FTDI)
  ┌─────────────────┐           ┌──────────────┐
  │ TX  ────────────┼──────────▶│ RX           │
  │ RX  ◀───────────┼───────────│ TX           │
  │ GND ────────────┼──────────▶│ GND (–)      │
  │ 5V  ────────────┼──────────▶│ VCC (+)      │  ← only if not powered separately
  └─────────────────┘           └──────────────┘

  ⚠️ TX ─▶ RX  and  RX ◀─ TX  (they CROSS over — common mistake!)
  ⚠️ Make sure both devices share the same GND connection

  Baud rate: 9600  |  8 data bits  |  No parity  |  1 stop bit  (8N1)
